> ## Penjelasan Singkat

Project ini di bangun untuk memfasilitasi client Baezeni untuk memngatur semua kontent di dalam website mereka.

> ## Project Goals

Dalam projek ini saya di tugaskan untuk membuat modul Content Management System, dengan module ini semua client Baezeni dapat mengatur konten mereka dengan mudah. saya bekerja untuk menghubungkan website BZ Publish dengan Api BZ Publish. dan membuat tampilan admin sesuai design dan flow yang sudah di tentukan.

> ## Bagaimana BZ PUblish bekerja ?

### Client & Templating setup

for the first step, each client have one website template, this template for collect client data and information, also automation will create stroge in azure store base on each clients template.
in this part the admin baezeni also setup module subscription and site domain

### Site Structure Setup

Site structure containing client sitemap, the forntend will create all page base on sitemap, in BZ Publish, client can create unlimited page

### Section Templating

Section Templating . In this part frontend will make all section for each page base on design, frontend will use BZ Publish CMS Module for fill the containing
and the consume the contain with api. client can pair unlimited section to page they want.
The section containing data BZ Publish Have 5 widget Data for acomodir all client need.

we could them widget

- **Article** for save article collection
- **Job Poster** for save Job Poster collection
- **Media** for save media collection
- **Text** for save Text collection
- **Table** for generate Table and Chart data

### Build website

the last step . actually this simple step , for build website admin just select build type , there are two options Preview and Publish, usually preview build the first, after all good with no bug, and then next build the publish

> ## The Problems and How I Deal With It

Ini adalah projek terbesar yang pernah saya kerjakan, dan banyak menggunakan teknologi baru pada projek ini.

BZ Publish telah di siapkan untuk memenuhi berbagai kebutuhan client, banyak hal yang di buat sangat dinamis. seperti : multipe template, multipe hak akses, multipe bahasa dan masih banyak lagi. tantangan yang sangat menarik untuk saya adalah menangani multiple bahasa. dalam setiap CRUD proses saya harus memvalidasi semua form dengan berbagai bahasa, dan memastikan semua data yang dikirm sesuai dengan ketentuan Backend. selama menangani kasus ini, saya sangat banyak belajar tentang Array handling method, handling http request dan banyak hal lain yang sangat berguna untuk saya pelajari dan kuasai.
